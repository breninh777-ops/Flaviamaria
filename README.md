# Site Psicóloga (landing page)

Abra o arquivo [`index.html`](./index.html) no navegador.

## Personalização rápida
- Foto do hero: coloque um arquivo em `site-psicologa/assets/hero.png`.
- Fundo (opcional): coloque um arquivo em `site-psicologa/assets/bg.jpg`.
- Nome/CRP/textos: edite direto em `site-psicologa/index.html`.
- Cores e fonte: `site-psicologa/styles.css` (variáveis em `:root`).
- WhatsApp/e-mail: seção **Contato** em `site-psicologa/index.html` e o `mailto:` em `site-psicologa/script.js`.

## Rodar com servidor local (opcional)
No PowerShell, dentro de `site-psicologa/`:

```powershell
python -m http.server 8000
```

Depois acesse `http://localhost:8000`.
